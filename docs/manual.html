2. Navigation & Search

zoxide (smart cd)

Alias Description
cd z — frecency-based directory jump
cdi zi — interactive fzf directory picker
.. Go up one directory
... Go up two directories
.... Go up three directories

fzf (fuzzy finder)

Shortcut Description
Ctrl+T Insert file path (fd backend, respects .gitignore)
Alt+C cd into directory (eza tree preview)
Ctrl+/ Toggle preview panel

atuin (history)

Daemon-backed in-memory search indexing enabled for fast results.

Shortcut / Command Description
Ctrl+R Full-text history search (replaces fzf Ctrl+R)
atuin ai "query" AI-powered command generation from natural language

ripgrep

Alias Description
rgi rg -i — case-insensitive search
rgf rg --files | rg — search filenames
rgl rg --files-with-matches — list matching files only

Custom fzf functions

Command Description
frg <pattern> rg + fzf + bat preview → opens in $EDITOR at line
fkill [signal] ps + fzf multi-select → kill processes (default signal 9)
Flow: Find text in code and open
frg pattern → browse matches with bat preview → select → opens in $EDITOR at line

5. Worktrees (Worktrunk)

Commands

Command Description
wt create <name> Create worktree + auto-install deps (bun/pnpm/npm)
wt remove <name> Remove worktree (syncs settings back first)
wt list List all worktrees
wt commit Commit with AI-generated message (Claude Haiku)
wt cd <name> Switch to worktree directory
wt step <alias> Run a custom step alias (e.g. wt step copy-ignored)
wt merge --no-ff Merge with semi-linear history (no fast-forward)

Hooks

Hook Description
pre-start Copies ignored files and saves base worktree path (blocking — aborts on failure)
post-start Installs deps via detected package manager (background — non-blocking)
pre-remove Syncs .claude/settings.local.json back to base via jq merge
Flow: Isolated feature work
wt create feat-x → auto deps install → code → wt commit (AI message) → merge
